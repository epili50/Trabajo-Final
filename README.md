# Trabajo-Final
Itinerario de Data Science IT Academy. Barcelona.

El objetivo de este proyecto es predecir si pacientes internados/ingresados en Terapia Intensiva estarán allí más de una semana, 
o antes de los siete días dejarán esa condición, ya sea por alta médica, defunción u otra causa. Esa predicción sehará en base a 
la edad del paciente, el sexo y las mediciones de signos vitales que se le toman al mismo durante los dos primeros días de internación.
